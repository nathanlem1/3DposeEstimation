### 3DposeEstimation
This code estimates a 3D pose given model and sample point clouds as input using 
[Open3D](http://www.open3d.org/) library.

### Dependencies
- Numpy
- Open3D

### Running Example Code
And you can run this module using:
```bash
python 3DposeEstimation.py
```
